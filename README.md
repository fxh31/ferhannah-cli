# ferhannah-cli

![ferhannah-cli version](https://img.shields.io/badge/ferhannah-cli_v1.0.0-green)

## 简介

用于快速搭建项目的命令行工具。

## 安装

```bash
npm install ferhannah-cli -g
```

## 使用

### 创建项目

```bash
ferhannah-cli create [project-name]
```

### 查看版本

```bash
ferhannah-cli -v
ferhannah-cli -version
```

### 更新到最新版本

```bash
ferhannah-cli update
```
